Slide 07 / Limitations

Honest Limitations

What I haven't proven yet.

1 Single repo. FastAPI only. Different structures may see different results.
2 Python only in this benchmark. JS, Go, Rust untested at scale.
3 Open-source embeddings. Cloud tools with larger models likely retrieve better on ambiguous queries.
4 Your codebase is not FastAPI. Your number will be different. I want to know what it is.

Reproduce It

2 commands. 60 seconds.

$ pip install code-context-engine

$ python benchmarks/run_benchmark.py \
    --repo https://github.com/fastapi/fastapi.git \
    --source-dir fastapi
